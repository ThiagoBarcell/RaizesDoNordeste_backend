# Raízes do Nordeste API

API REST desenvolvida em **Delphi 10.3 Rio** com o framework **Horse**, utilizando **PostgreSQL** como banco de dados.
---

## Objetivo do projeto

Construir uma API backend para uma rede de restaurantes com foco em:

- cadastro de usuários
- autenticação com JWT
- persistência em PostgreSQL
- organização do projeto em camadas

---

## Tecnologias utilizadas

- Delphi 10.3 Rio
- Horse
- Jhonson
- Horse JWT
- BCrypt
- FireDAC
- PostgreSQL
- BOSS
